Where to Begin

Start Here

Five amplifier skills that set up everything else. Run these before diving into the library — they create the reference files, configure the memory, and establish the strategic foundation that all downstream skills draw from.

Amplifiers — Run These First

These produce output that other skills consume. Run them in order if you're starting fresh.

1
/motivation-spectrum-analyzer
Run this first on any new market. It scores your market on the Away/Toward/Hybrid spectrum and tells you which of the three demand-creation paths to use. Everything downstream — the excavation, the concept, the USP — runs differently depending on this score.
The three demand paths produce different outputs. Running the wrong path wastes all the work that follows.
2
/personal-brand-dna-extractor
Extracts voice patterns, credential structure, stories, and philosophy from live social feeds and produces 8 brand files. These files get referenced by the copywriting, webinar, and content systems — so every output sounds like you, not like Claude.
Run this once. Every other skill that requires 'your voice' reads from these files.
3
/setup-memory-system
Installs persistent memory so Claude Code remembers your clients, projects, preferences, and prior work across sessions. Without this, every conversation starts cold.
Most of the advanced skills get dramatically better when they can access prior context about your market, clients, and work.
4
/psychographic-excavation
The foundational market research skill. Excavates your market's beliefs, rage points, secret aspirations, identity statements, failed solutions, and raw language — the kind that shows up in 3am Reddit posts, not survey responses. Everything else builds on this.
The demand pipeline, offer design, and copywriting all draw from this. Skip it and everything downstream is built on assumptions.
5
/distinction-finder
Extracts genuine intellectual distinctions from expert content — the things you know that others don't, that, when stated correctly, create a perceptual shift. Feeds framework design, positioning, and authority content.
Distinctions are the raw material of frameworks, core concepts, and positioning. You can't package expertise you haven't extracted.
